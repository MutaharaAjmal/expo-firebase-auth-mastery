# 🔐 Expo Firebase Auth & Firestore CRUD Mastery

A full-stack mobile starter built with **React Native (Expo)** featuring Firebase services, advanced navigation, and mock API integration.

## 🌟 Key Features
- **Firebase Auth:** Email/Password Login, Signup, Reset Password, and **Google Sign-In**.
- **Firestore CRUD:** Full User Management (Create, View, Edit, and Delete users directly in Firestore).
- **State Management:** Counter & User state handled via **Zustand**.
- **Navigation:** Modern file-based routing with `expo-router`.
- **API Integration:** Data fetching using `json-server`.
- **UI Components:** Custom Buttons, Modals, AsyncStorage for persistence, and Platform-specific UI.

## 🚀 Setup & Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/MutaharaAjmal/expo-firebase-auth-mastery.git](https://github.com/MutaharaAjmal/expo-firebase-auth-mastery.git)
   Install dependencies:

2. **Install dependencies:**

npm install
3. **Run Mock API (json-server):**
To see the API integration in action, run:

npx json-server --watch db.json --port 3000
4. **Run the Expo App:**

npx expo start
